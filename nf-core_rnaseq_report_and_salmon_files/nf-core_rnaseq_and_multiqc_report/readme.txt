##README
When the 'GRCm39_109_ensembl_reference' directory is referenced in the nf-core/RNAseq reports, that directory contained Mus_musculus.GRCm39.109.gtf.gz and Mus_musculus.GRCm39.dna.primary_assembly.fa.gz which were obtained from the ENSEMBL database and was the Mus muscuslus reference assembly used for this study.
